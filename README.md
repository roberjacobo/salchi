**A healthy life for your best friend**

Salchi is an AI-powered hub dedicated to everything Dachshund. Our mission is to help Dachshund owners provide the best possible care for their beloved companions. Whether you're a new owner or a seasoned Dachshund enthusiast, Salchi offers comprehensive information about breed characteristics, care tips, health considerations, and more.

## About the Project

This web application provides valuable resources for Dachshund owners, including:

- **Breed Information**: Learn about the origin, history, and unique characteristics of Dachshunds
- **Care Tips**: Essential guidance on exercise, grooming, nutrition, and health
- **Breed Varieties**: Explore different types including Standard, Miniature, and various coat types
- **Health Focus**: Special emphasis on back care and other health considerations specific to Dachshunds

Built with modern web technologies to provide a smooth, responsive experience across all devices.

## Tech Stack

This project was built using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.3, featuring:

- Angular 21 with standalone components
- Server-Side Rendering (SSR)
- Tailwind CSS for styling
- TypeScript in strict mode
- Vitest for testing

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
